# 경쟁 제품 학습 — PTI 취업직무코드 프로파일 (마인드코딩)

> 출처: PTI 코칭센터 결과지 7부 (개인별 동일 템플릿, Basic 버전, 각 ~20p)
> 발행: 주식회사 마인드코딩 / MC-Profile Ver 1.2.64 / © 2023
> 목적: 구조·깊이·서술 방식을 **학습**해 Ahee Career Code 결과지를 업그레이드하기 위한 분석.
> ⚠️ IP 경계: PTI의 **CODE 4유형(Catalyst/Organizer/Diverger/Empath)·16유형 명칭·체계는 마인드코딩의 상용 자산 → 복제 금지.** Ahee는 **공개 도메인인 RIASEC만** 사용. 아래는 "구조·서술 패턴"만 학습 대상.

## 1. 제품 정체성

- **이름:** 취업직무코드 프로파일 (Career Fit & Job Performance CODE Profile)
- **핵심 프레임:** PTI(자체 16유형) **+ 홀랜드 RIASEC + NCS 직업기초능력** 3종 결합
  - PTI 16유형 → 일하는 방식·행동 구조 → 직무 강점·조직 적응 전략
  - RIASEC → 직업흥미·선호 분야 → 관심 직무·진로 탐색
  - NCS → 채용시장 요구 역량 → 자소서·면접 역량 표현·보완
- **톤:** 청록(teal) 그라데이션 + 화이트, B2B 코칭센터용 정장 느낌. 모든 페이지 동일 헤더(장 번호 + 유형 배지)·푸터(저작권·페이지·이름).
- **티어 구조:** Basic(이 결과지, 자기이해+기본전략) → **Pro 버전 업셀**(10가지 상황별 심화·전문가 상담용).

## 2. 전체 목차 (6장 + 프롤로그 + 에필로그)

| 구획 | 내용 |
|---|---|
| **표지** | 이름·코드(검사ID)·검사기관·전문가, QR 2개(프로파일 정보 / 내담자 프로파일) |
| **프롤로그** | "취업은 회사 고르기가 아니다" 동기 → 이 프로파일이 답하는 질문 4개 → PTI란? CODE란?(4유형 표) → 누구에게 필요한가 → PTI·RIASEC·NCS 함께 보는 이유(비교표) → 활용법 |
| **1장 진단** | 16유형에서 내 자리 → 작동코드 설명 → 안정코드 설명 → 한 줄 정의 → **RIASEC 매칭(육각형 그림 + 끌리는일/통하는점/대표직업 표)** → **NCS 매칭(강한 능력 3 / 보완할 능력 3)** → **최강점 영역 5카테고리**(각 대표직업·핵심 자격증·잘 맞는 기업) → 채용 전략 한 줄 |
| **2장 작동·안정의 흐름** | 강점(정렬 시 발현) 표 → 그림자(위축 시 발현) 표 → "강점과 그림자는 같은 힘" 진실 → 작동·안정 만남 + 정렬될 때/어긋날 때 비교표 |
| **3장 직무선택** | 일상에서 나타나는 모습(따옴표 구어체 5개) → 직무선택 흐름(결정 방식·잘 맞는 직무 방향·무너지는 지점) |
| **4장 업무성향/자기표현** | 업무 실행·관계 협업 방식, 자소서·면접에서 강점 표현법 |
| **5장 조직 적응** | 입사 후 관계·피드백·업무 리듬 적응 |
| **6장 커리어 설계** | 이직·전환·승진·전문성 강화 방향 |
| **에필로그** | 전체 요약 → Basic vs Pro 비교표 → Pro에서 더 보는 것 → "이런 경우 Pro 권함" |

## 3. 학습 포인트 (Ahee에 적용 가능한 "구조·서술" 패턴)

1. **3종 렌즈 결합 서사** — 흥미(RIASEC) + 일하는 방식 + 채용 역량(NCS)을 한 결과지에서 교차. Ahee는 RIASEC 1종이라 깊이가 얕음 → **NCS 직업기초능력 매칭**은 공개 표준이라 합법적으로 추가 가능(자소서·면접 연결 강화).
2. **강점 ↔ 그림자(주의점)를 "같은 힘의 양면"으로** 서술. 단순 장단점 나열보다 설득력 큼.
3. **"일상에서 이런 모습으로 나타나요"** — 따옴표 구어체로 자기 인식 유도("관심 분야가 너무 많아 어디부터…"). 공감 후 처방 구조.
4. **최강점 영역 = 직무 카테고리별 (대표직업 · 핵심 자격증 · 잘 맞는 기업)** 3종 세트. Ahee의 "직무 매칭 TOP5"보다 실행정보 풍부 → **자격증·기업 예시**를 붙이면 학생 활용도 급상승.
5. **취업 4단계 챕터화** (직무선택 → 자소서·면접 → 입사 후 적응 → 커리어 설계). Ahee는 "Action Guide" 1페이지뿐 → 단계별로 확장 여지.
6. **Basic→Pro 티어** = 무료 결과지로 가치 입증 후 심화 유료. Ahee의 빠른검사→정밀검사, 결과지→상담 업셀에 그대로 응용 가능.
7. **RIASEC 육각형 시각화 + 1·2·3위 뱃지** — Ahee 결과지의 막대그래프 대비 직관적. 육각형 추가 검토.

## 4. 의도적으로 베끼지 않을 것 (IP/톤)
- CODE 4유형·16유형 명칭·정의·"작동/안정코드" 용어 체계 (마인드코딩 자산).
- 청록 톤·정장 B2B 레이아웃 (Ahee는 마젠타 에디토리얼 톤 유지).
- "PTI" 등 브랜드·검사명.
