section 03 ● key decision points

핵심 분기 — 누구를 어디로 보낼 것인가

단순 5단계 일직선이 아닙니다. 사용자 의도와 채널에 따라 다른 흐름이 동시에 굴러갑니다. 가장 중요한 두 분기는 "퀵 vs 정규"와 "개인 vs 학교 단위"입니다.

● 분기 01

입구는 다르지만 결국 만나는 두 길

QUICK PATH
v0.3 카드형 (24문항)
인스타·부스에서 진입. 가볍게 받고 공유. 호기심·바이럴 입구. 전환율 25-35%를 v0.2로 끌어올림.
FULL PATH
v0.2 정규형 (100문항)
학교 진로 프로그램·정규 수업·상담 연계로 진입. 진지한 사용자. 완료율 80%+ 목표. 자소서·면접 코칭으로 연결.
● 분기 02

고객은 학생 한 명? 학교 단위?

B2C
개인 학생
무료 카드 검사 → 정규 검사 (2만원) → 자소서 코칭 (5만원). LTV 7만원. SNS 바이럴 핵심.
B2B
학교·기관 단위
진로 프로그램 패키지 (학생 100명 기준 200-500만원). 대학별 데이터 리포트 추가 옵션. 핵심 수익원.
section 04 ● ahee backend flow

학생이 안 보는 영역 — 운영 흐름

학생 데이터가 들어와서 비즈니스 가치로 전환되는 4단계 + 산출물. 이게 Ahee의 진짜 자산이 됩니다.

01
데이터 수집
학생 응답 + 메타데이터 (학교·전공·학년)를 Supabase에 익명/식별 옵션으로 저장.
→
02
자동 분석
RIASEC 채점 알고리즘 + 페르소나 매칭 → 결과 카드 PDF 자동 생성 (Playwright).
→
03
집계·통계
학교별·전공별·시기별 페르소나 분포 데이터 축적. 트렌드 분석 가능.
→
04
후속 제안
개인: 코칭·자소서 / 학교: 학생 진로 데이터 리포트 · 다음 학기 프로그램 제안.
→
OUTPUT
비즈니스 가치
학교 B2B 제안서 / 개인 결제 / 학생 데이터 자산화 — 3중 수익 구조.

★
strategic takeaway ● 핵심 3가지

이 흐름이 작동하려면 3가지가 반드시 있어야 합니다.

01
바이럴 가능한 결과 카드
v0.3 카드는 단순한 결과지가 아니라 "공유하고 싶은 콘텐츠"여야 합니다. 디자인이 모든 깔때기의 출발점.
02
B2C에서 B2B로의 다리
개인 학생 수익만으로는 사업이 안 됩니다. 학교 단위 패키지가 메인 수익. 개인은 신뢰 확보와 데이터 축적 채널.
03
데이터의 자산화
매 검사가 새로운 데이터 포인트. 1년 후 "전국 대학생 페르소나 트렌드 리포트"가 됩니다. 그 자체가 제품.
